# UIU TalentHUB

UIU TalentHUB is a specialized digital ecosystem developed for United International University students to showcase creative and technical work across audio, video, and editorial categories.

## Technical Stack

- **Backend**: PHP 8.x
- **Database**: MySQL
- **Frontend**: HTML5, Vanilla CSS3, Vanilla JavaScript
- **Libraries**: Wavesurfer.js (Audio Visualization), Quill.js (Rich Text Editor)
- **Environment**: Optimized for XAMPP / Apache

## Core Features

- **Unified Upload Hub**: Streamlined interface for publishing audio tracks, video performances, and rich-text journals.
- **Advanced Audio Experience**: Interactive waveform visualization and real-time category filtering (Music, Poetry, Podcasts, Stories).
- **Dynamic Video Bento Grid**: Responsive layout featuring intelligent aspect-ratio adaptation and hover-triggered playback.
- **Personalized Student Profiles**: Automated identity generation using color-coded avatars and initials based on student credentials.
- **Secure Authentication**: Multi-stage registration system with mandatory @uiu.ac.bd domain enforcement and OTP verification.
- **Content Management**: Secure deletion API and engagement tracking (Likes) across all media types.

## Installation

1. Clone the repository to your local web server directory (e.g., `htdocs`).
2. Import the database schema from `database/uiu_talenthub.sql`.
3. Configure database credentials in `includes/db_connect.php`.
4. Ensure the `uploads/` directory has appropriate write permissions.
